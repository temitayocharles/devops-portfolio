Ansible Packages Role

Designed and implemented an advanced Ansible role to manage installation, upgrades, and cleanup of system packages across Linux environments.

Features

  • Conditional logic for platform-aware execution (apt/yum)
  • Retry mechanism for transient package errors
  • Version pinning support
  • Logging and error handling with fail-safe defaults

Stack: Ansible, Bash, YAML, Ubuntu, Amazon Linux
