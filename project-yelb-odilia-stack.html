üîÑ Disaster Recovery Sequence (RTO: <30s)

2. DNS Failover (15s): Route 53 switches traffic to Region 2 standby endpoints
3. Promotion (5s): Promote PG Replica 3 to PRIMARY, scale up app servers to 3 replicas
4. Traffic Flows (Total: <30s): Users now connected to Region 2, zero data loss (sync replication)
