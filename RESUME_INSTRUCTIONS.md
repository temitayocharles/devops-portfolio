## Converting HTML to PDF

### Method 1: Using Chrome/Edge Browser (Recommended)
1. Open the resume HTML file in Chrome or Edge browser
2. Press `Ctrl+P` (Windows) or `Cmd+P` (Mac) to open Print dialog
3. Select "Save as PDF" as the destination
4. **For ATS Resume:**
   - Set margins to "Default"
   - Enable "Background graphics"
   - Click "Save"
   
5. **For Visual Resume:**
   - Set margins to "None" or "Minimum"
   - Enable "Background graphics"
   - Click "Save"

### Method 2: Using Safari (Mac)
1. Open the resume HTML file in Safari
2. Press `Cmd+P` to open Print dialog
3. Click "PDF" dropdown at bottom-left
4. Select "Save as PDF"
5. Choose location and save

### Method 3: Online Conversion (Alternative)
1. Visit any HTML to PDF converter (e.g., https://www.web2pdfconvert.com/)
2. Upload the resume HTML file
3. Download the generated PDF

## Best Practices for Job Applications

### When to Use ATS-Friendly Resume
- Applying through company career portals
- Submitting to job boards (Indeed, LinkedIn Jobs)
- Any application that mentions "ATS" or "keyword matching"
- Large corporations with automated screening

### When to Use Visual Resume
- Email applications directly to hiring managers
- Networking events and conferences
- Portfolio presentations
- LinkedIn profile downloads
- Personal website links
